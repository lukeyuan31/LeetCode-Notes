###  反转二叉树
伪代码：

```
void invertTree(Node root){
    if (root){
        swap left,right;
    }
    else {
        return root;
    }
    
    reverse(left);
    reverse(right);
}

```
遍历二叉树方法（递归先序）

```
void  reverse(Node root){
    if(root){
        access root;
    }
    if (left){
        reverse(left);
    }
    if (right){
        reverse(right);
    }
}
```
注意分辨，由于反转二叉树时需要考虑null的情况，此时不用添加判断语句，遍历时由于不用访问null，需要添加判断语句防止空指针exception
