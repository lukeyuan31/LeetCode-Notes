# LeetCode学习笔记

Time: 2019/08/23-
Language: Java

## Easy Mode
### Java subString()功能：

```
public String substring(int beginIndex, int endIndex)
```

第一个int为开始的索引，对应String数字中的开始位置，  
第二个是截止的索引位置，对应String中的结束位置

### Java charAt()功能

检索在i处的character，可以用于string类型

注意charAt()所得到的结果为Char类型，比较时用单引号，与String区分，留心数据类型的变换
